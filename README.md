# Hospital Management System

A comprehensive hospital management system built with React and Tailwind CSS.

## Features

- **Patient Wallet System** - Fund management and transaction tracking
- **Patient Management** - Complete patient records and admission management
- **Laboratory & Radiology** - Test results management and file uploads
- **Doctor's Dashboard** - Patient history, diagnosis, and prescription management
- **Pharmacy Management** - Prescription processing and medication dispensing
- **Admin Panel** - Staff management and system reporting
- **Discharge Module** - Service completion verification and billing
